# NOTE: numpy is pinned below 2.0 because spaCy's "thinc" dependency ships
# precompiled Cython extensions built against numpy 1.x's internal memory
# layout. numpy 2.0 changed that layout, causing a
# "ValueError: numpy.dtype size changed, may indicate binary incompatibility"
# crash at import time. This is a known, widely-reported spaCy/thinc issue.
#
# spacy is also pinned below 3.8.0 to match the en_core_web_sm 3.7.1 model
# wheel above (a spaCy model's major.minor version must match the spaCy
# version that loads it).
#

# only installs from this file - it won't run "python -m spacy download").
